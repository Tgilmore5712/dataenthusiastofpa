# dataenthusiastofpa.com

Scaffolded Next.js + Tailwind site for dataenthusiastofpa.com.

## Local setup

Install dependencies and run dev server:

```bash
cd dataenthusiastofpa
npm install
npm run dev
```

Open http://localhost:3000 to view.

## Deploy

I recommend deploying to Vercel: connect the repo to Vercel, set the root to the project, and it will build automatically.
